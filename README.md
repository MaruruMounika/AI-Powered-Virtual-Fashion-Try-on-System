# AI Powered Virtual Fashion Try-On System

A Flask-based AI virtual try-on web application where users can upload their image and preview garments digitally.

## Features
- User Login & Signup
- Garment Selection
- Image Upload
- Virtual Try-On Generation
- Flask Backend
- SQLAlchemy Database
- Render Deployment

## Technologies Used
- Python
- Flask
- SQLAlchemy
- HTML
- CSS
- Pillow (PIL)
- Render
